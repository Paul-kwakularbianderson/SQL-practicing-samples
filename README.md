# SQL-practicing-samples
## Collection of my sample SQL files
This is a collection of sample SQL solutions in a Bootcamp by WBS coding school.

### Libraries used
querying, filtering, aggregating, and joining relational database tables

### Key Learning
- Practiced writing multi-table SQL queries using JOIN, GROUP BY, HAVING, and CASE WHEN statements to answer specific business questions from raw relational data.
- Learned how to use Tableau's drag-and-drop interface to build bar charts, line graphs, heat maps, and filters 
- turning query outputs into compelling visual stories.
### Challenges Overcome
- Slow query performance: Some queries on large tables were running slowly.
- I learned to optimize them by selecting only necessary columns, avoiding SELECT *, and using WHERE clauses to filter early in the query.
- Tableau formatting issue was a big challenge.
  - Getting charts to display correctly for different audience screen sizes took trial and error.
  - I learned to use Tableau's fixed layout vs. automatic sizing options and to test dashboard views at different resolutions.
